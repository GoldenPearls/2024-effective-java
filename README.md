## 🥮 진행방식

### 📌 스터디 전

- 정해진 분량을 읽고 각자 내용 정리

  - 공유 방식: 주차별 깃허브 이슈에 댓글로 자료 url 남기기
  - 불참시에도 자료 공유는 필수 ⭐️

### 📌 스터디 당일

- 랜덤으로 2명의 발표자 선정
